reCAPTCHA is an extension that replaces ExpressionEngine's built-in CAPTCHA with the reCAPTCHA system on member registration and comment forms.

Installation

  1. Upload the recaptcha folder to system/expressionengine/third_party
  2. Enable and configure the extension via Add-Ons > Extensions
